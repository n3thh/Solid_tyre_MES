# 🐧 3️⃣ Operating System Hardening (Ubuntu Server)

Install:

```bash
Ubuntu Server LTS (minimal)
```

Then secure it:

## 🔐 Step 1 – Create Non-Root Admin

```bash
adduser factoryadmin
usermod -aG sudo factoryadmin
```

Disable direct root login.

---

## 🔥 Step 2 – Enable Firewall (UFW)

```bash
sudo ufw allow 22
sudo ufw allow from 192.168.0.0/24 to any port 5432
sudo ufw enable
```

Block everything except LAN + SSH.

---

## 🚫 Step 3 – Disable External Access

In `/etc/postgresql/*/main/postgresql.conf`

```ini
listen_addresses = '192.168.0.141'
```

In `pg_hba.conf` allow only LAN subnet.

No internet exposure.

# 🗄 4️⃣ Database Security Architecture

## 🚨 CRITICAL: Stop Using postgres Superuser

Create dedicated DB role:

```sql
CREATE USER factory_app_user WITH PASSWORD 'STRONG_PASSWORD';
GRANT CONNECT ON DATABASE tyre_factory_db TO factory_app_user;
GRANT USAGE ON SCHEMA public TO factory_app_user;
GRANT SELECT, INSERT, UPDATE ON ALL TABLES IN SCHEMA public TO factory_app_user;
```

Never use postgres in application.

---

## 🔑 Password Storage Security

In `users` table:

Replace plain passwords with:

```python
bcrypt.hashpw(password.encode(), bcrypt.gensalt())
```

Never store raw passwords again.
